Project 1:🏡 Linear Regression Model for House Price Prediction

This project uses a Linear Regression model to predict house prices based on median income. By analyzing housing-related data, it establishes a relationship between income levels and property prices, helping users estimate housing costs.

🔹 Key Features:

  • ✅ Data-Driven Predictions: Uses real housing data to make price estimates.
  • ✅ Interactive Web App: Built with Streamlit for real-time predictions.
  • ✅ Line Chart Visualization: Displays how house prices change with different incomes.
  • ✅ Web Deployment: Accessible via GitHub Pages / Streamlit Cloud.

🔹 Technologies Used:

  • Programming Language: Python
  • Machine Learning: Linear Regression (via scikit-learn)
  • Libraries: pandas, numpy, streamlit, matplotlib
  • Deployment: GitHub + Streamlit Cloud

🔹 How It Works:

  1. User Input: Users enter their median income.
  2. Model Prediction: The trained model predicts the house price.
  3. Visualization: A line chart shows price variations.
  4. Results: The predicted price is displayed.
