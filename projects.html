🏡 Linear Regression Model for House Price Prediction

This project leverages a Linear Regression model to predict house prices based on the median income of a region. By training the model on a dataset containing housing-related features, it establishes a relationship between income levels and property prices, allowing users to estimate housing costs based on income inputs.

🔹 Key Features:

  • ✅ Data-Driven Predictions: Uses historical housing data for reliable price estimates.
  • ✅ Simple & Effective UI: Built using Streamlit for real-time predictions.
  • ✅ Interactive Visualization: Includes a line chart showing predicted house prices.
  • ✅ Web Deployment: Accessible online via GitHub Pages / Streamlit Cloud.

🔹 Technical Details:

  • Model Used: Linear Regression
  • Programming Language: Python
  • Libraries Used:
    • 📌 scikit-learn (for training the model)
    • 📌 pandas & numpy (for data processing)
    • 📌 streamlit (for building the interactive web app)
    • 📌 matplotlib (for visualization)
  • Deployment: Hosted via GitHub + Streamlit Cloud.

🔹 How It Works:

  1. User Inputs: The user enters their median income into the app.
  2. Model Prediction: The trained model predicts the house price based on income.
  3. Visualization: A line chart displays how house prices vary with income.
  4. Results: The predicted price is displayed for user analysis.
